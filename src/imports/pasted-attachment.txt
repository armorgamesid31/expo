Revise and upgrade the Salon Management Mobile SaaS UI to align with a real production SaaS architecture.

Important context:
This system is not just a booking app. It includes:
- AI WhatsApp booking agent
- Automatic website builder
- Customer risk detection system
- Multi-language SEO website
- Multi-tenant architecture

The mobile app should function as the **Salon Operating System** for owners and staff.

Improve the product with the following upgrades.

Owner / Admin
Specialist / Staff
